# 视觉风格分类与决策树

> `format-style-planner` Step 3 读取本文件，按决策树匹配最佳视觉风格。

---

## 风格分类

### A — 2D 插画 / 动态漫画

**子类型：**
- **赛璐璐风 (Cel-shading)**：硬边阴影、色块分明、日漫感
- **厚涂风 (Painterly)**：笔触可见、层次丰富、韩漫/国漫感
- **扁平风 (Flat)**：极简色块、无明显阴影、现代设计感
- **水彩/手绘风**：透明叠色、自然渗透、文艺感
- **漫画网点风**：黑白或少色、网点覆盖、日式漫画感

**适用场景：** 情绪夸张的题材、风格化强的作品、产能要求高的项目
**制作优势：** 生成速度快、风格一致性较容易维持、不受恐谷效应影响
**制作限制：** 动态表现力依赖后期动效、写实感较弱

### B — 3D 渲染

**子类型：**
- **写实 CG (Photorealistic CG)**：接近真人的渲染质量、影视级光影
- **半写实 (Semi-realistic)**：介于写实和卡通之间、游戏 CG 感
- **卡通 3D (Stylized 3D)**：3D建模但卡通化渲染、皮克斯风

**适用场景：** 需要空间感和光影表现的题材（悬疑/科幻/都市）
**制作优势：** 空间感强、镜头运动丰富、角色表演连贯
**制作限制：** 成本高、一致性控制难度大、恐谷效应风险

### C — 真人仿真 3D

**子类型：**
- **AI 真人仿生 (AI Photoreal)**：即梦等工具生成的仿真人像
- **MetaHuman/数字人**：UE 引擎驱动的超写实数字人
- **照片合成**：AI 生成照片级画面、类真人短剧效果

**适用场景：** 都市情感、职场、悬疑、想要"真人剧"观感的项目
**制作优势：** 观众接受度高（像真人剧）、沉浸感强
**制作限制：** 恐谷效应是最大风险、角色一致性极难维持、手部/细节容易穿帮

### D — 混合风格

**组合方式：**
- 2D 角色 + 3D 背景
- 3D 主场景 + 2D 特效/回忆
- 真人仿真主线 + 风格化插叙

**适用场景：** 有多种叙事层次的作品（现实+幻想、现在+过去）
**制作限制：** 风格切换节点需精心设计，否则观感割裂

## 决策树

按以下顺序排查，选择第一个匹配的路线：

```
Q1: 项目追求"真人短剧"观感？
├── 是 → 路线 C（真人仿真3D）
│   └── Q1a: 团队能处理恐谷效应问题？
│       ├── 是 → 确认路线 C
│       └── 否 → 降级到路线 B（半写实3D）
│
└── 否 → 继续
    Q2: 题材是否强依赖情绪夸张/表情放大？
    ├── 是 → 路线 A（2D插画）
    │   └── Q2a: 题材偏日系？
    │       ├── 是 → A-赛璐璐风
    │       └── 否 → A-厚涂风 或 A-水彩风
    │
    └── 否 → 继续
        Q3: 题材是否需要强空间感/光影叙事？
        ├── 是 → 路线 B（3D渲染）
        │   └── Q3a: 写实感重要还是风格化重要？
        │       ├── 写实 → B-写实CG 或 B-半写实
        │       └── 风格化 → B-卡通3D
        │
        └── 否 → 继续
            Q4: 产能约束如何？
            ├── 需要快速/低成本 → 路线 A（扁平风或赛璐璐风）
            ├── 中等 → 路线 A（厚涂风）或 路线 B（半写实）
            └── 充裕 → 路线 B 或 路线 C
```

## 题材-风格匹配参考

| 题材 | 推荐风格 | 理由 |
|------|----------|------|
| 甜宠/恋爱 | A-厚涂 或 C-真人仿真 | 厚涂适合梦幻感，真人仿真适合代入感 |
| 逆袭/爽文 | A-赛璐璐 或 C-真人仿真 | 需要节奏快、表情夸张 |
| 古装/仙侠 | A-厚涂 或 B-半写实 | 需要画面美感和氛围 |
| 都市/职场 | C-真人仿真 或 B-半写实 | 写实感增强可信度 |
| 悬疑/惊悚 | B-写实CG 或 C-真人仿真 | 光影叙事关键 |
| 玄幻/热血 | A-赛璐璐 或 B-卡通3D | 动作表现力和夸张感 |
| 搞笑/日常 | A-扁平 或 A-赛璐璐 | 轻松感 |
| 科幻 | B-写实CG | 空间感和技术感 |

## 风格关键词速查

供 project-spec 的 `visual_keywords` 字段使用：

| 风格 | 关键词示例 |
|------|-----------|
| A-赛璐璃 | anime style, cel shading, vibrant colors, clean lines |
| A-厚涂 | digital painting, rich brushwork, warm tones, soft lighting |
| A-水彩 | watercolor, translucent, pastel, ethereal |
| B-写实CG | photorealistic 3D, cinematic lighting, ray tracing, volumetric |
| B-半写实 | semi-realistic, game cinematic, stylized realism |
| C-真人仿真 | hyperrealistic, photo quality, natural skin texture, studio lighting |
