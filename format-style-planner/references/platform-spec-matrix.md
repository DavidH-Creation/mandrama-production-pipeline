# 平台规格速查表

> `format-style-planner` Step 2 读取本文件，查目标平台的技术规格。

---

## 竖屏平台（9:16）

### 抖音

| 参数 | 规格 |
|------|------|
| 推荐分辨率 | 1080×1920 |
| 画幅比例 | 9:16 |
| 时长限制 | 15s / 60s / 3min / 10min（根据账号等级） |
| 漫剧常用时长 | 60-180s |
| 安全区-顶部 | 上方 120px 避开状态栏 |
| 安全区-底部 | 下方 300px 避开互动按钮区（点赞/评论/转发） |
| 安全区-右侧 | 右侧 100px 避开头像和按钮 |
| 字幕位置 | 底部居中，距底边 350-400px |
| 封面尺寸 | 1080×1920（与视频一致） |
| 格式 | MP4，H.264，推荐码率 6-8 Mbps |

### 快手

| 参数 | 规格 |
|------|------|
| 推荐分辨率 | 1080×1920 |
| 画幅比例 | 9:16 |
| 时长限制 | 57s / 10min（根据账号） |
| 安全区-底部 | 下方 280px |
| 安全区-右侧 | 右侧 90px |
| 字幕位置 | 底部居中，距底边 320-380px |

### YouTube Shorts

| 参数 | 规格 |
|------|------|
| 推荐分辨率 | 1080×1920 |
| 画幅比例 | 9:16 |
| 时长限制 | ≤60s |
| 安全区-底部 | 下方 200px 避开标题和按钮 |
| 字幕位置 | 底部居中，距底边 250-300px |
| 注意 | 超过 60s 不进入 Shorts 信息流 |

### 微信视频号

| 参数 | 规格 |
|------|------|
| 推荐分辨率 | 1080×1920 |
| 画幅比例 | 9:16（也支持 6:7, 16:9） |
| 时长限制 | 最长 30min |
| 漫剧常用时长 | 60-300s |
| 安全区-底部 | 下方 250px |
| 字幕位置 | 底部居中，距底边 300px |

## 横屏平台（16:9）

### B站

| 参数 | 规格 |
|------|------|
| 推荐分辨率 | 1920×1080（支持 4K） |
| 画幅比例 | 16:9 |
| 时长限制 | 基本无限制 |
| 漫剧常用时长 | 3-10min |
| 弹幕遮挡 | 上方 1/3 区域弹幕密集 |
| 字幕位置 | 底部居中，距底边 80-120px |
| 封面尺寸 | 1146×717 或 1920×1080 |

### YouTube（长视频）

| 参数 | 规格 |
|------|------|
| 推荐分辨率 | 1920×1080（支持 4K） |
| 画幅比例 | 16:9 |
| 时长限制 | 15min（默认），可申请更长 |
| 字幕位置 | 底部居中 |
| 封面尺寸 | 1280×720 |

---

## 方屏（1:1）

| 参数 | 规格 |
|------|------|
| 推荐分辨率 | 1080×1080 |
| 适用场景 | Instagram Feed、微博 |
| 注意 | 方屏在竖屏信息流中显示面积较小，不推荐作为漫剧首选 |

---

## 选择建议

| 场景 | 推荐画幅 |
|------|----------|
| 国内短剧平台（抖音/快手） | 9:16 竖屏 |
| 国内中长视频（B站） | 16:9 横屏 |
| 出海短视频（YouTube Shorts/TikTok） | 9:16 竖屏 |
| 出海中长（YouTube） | 16:9 横屏 |
| 微信视频号 | 9:16 竖屏 |
| 多平台分发 | 以主平台为准，裁剪适配次要平台 |
