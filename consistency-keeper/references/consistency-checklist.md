# Consistency Checklist

Use this checklist to score whether a generated sequence stays visually stable enough for the audience to accept it as one continuous piece of production.

## 1. Identity Consistency

Check the same character across adjacent shots:

- face shape
- hair style and hair color
- eye shape / iris color
- body build
- age appearance
- signature accessories

Severity guide:

- `minor`: recognizably same character, but one small detail drifts
- `major`: same role is still inferable, but drift is obvious on review
- `critical`: audience may read this as a different person

## 2. Costume And State Continuity

Check whether the correct outfit and state appear at the correct story moment:

- same outfit across same scene unless the story explicitly changes it
- damage, blood, dirt, tears, or wetness persist correctly
- hairstyle does not reset after a disruption event

## 3. Prop Continuity

Check all plot-relevant props:

- prop remains in hand when it should
- prop color and silhouette remain stable
- broken or opened props stay broken or opened
- prop scale does not drift between cuts

## 4. Environment Continuity

Check scene continuity:

- lighting direction remains coherent
- location layout does not jump
- key background elements stay present
- time-of-day or weather does not shift without story cause

## 5. Style Continuity

Check whether the sequence still feels like one project:

- line quality or texture
- realism level
- color palette
- contrast level
- lens/camera feel

## Verdict Rules

- `stable`: no critical issues, and no more than 1 minor issue per sequence
- `watch`: no critical issues, but several minor or 1 major issue exists
- `broken`: any critical issue, or multiple major issues that break continuity
