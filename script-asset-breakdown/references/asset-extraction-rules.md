# 资产提取规则与分类标准

> `script-asset-breakdown` Step 2 读取本文件，了解资产提取的分类标准和注意事项。

---

## 提取原则

1. **只提取需要视觉化的资产** — 对白中提到但从未出现在画面中的事物不算资产
2. **主角优先、龙套从简** — 主角需要完整档案，龙套只需基本轮廓
3. **跟踪状态变化** — 同一资产在不同场景可能有不同状态，必须记录变化点
4. **区分剧本描述和推断补全** — 明确标注哪些信息来自原文、哪些是推断

---

## 角色提取规则

### 必须提取的字段

| 字段 | 说明 | 补全策略 |
|------|------|----------|
| 名字 | 角色姓名/称呼 | 原文必有 |
| 角色类型 | 主角/配角/反派/龙套 | 根据出场频率和剧情权重判断 |
| 性别 | 男/女/其他 | 原文通常明确 |
| 外观年龄 | 看起来几岁 | 原文若无，从身份推断（大学生≈20、CEO≈35） |
| 发型发色 | 样式+颜色+长度 | 原文若无，根据角色性格和题材推断 |
| 体型 | 纤细/匀称/壮硕等 | 原文若无，从职业和性格推断 |
| 标志性特征 | 疤痕/眼镜/纹身/配饰 | 原文明确提到的优先，其余可推断 |

### 服装提取

- 每个角色至少记录一套"默认服装"
- 剧情中明确提到的换装必须记录为独立服装条目
- 服装描述应包含：类型（西装/校服/古装）、颜色、材质感、关键配饰
- 如果角色有"职业"属性但剧本未描述服装，按职业合理推断

### 表情/姿态

- 记录角色在剧中的常见表情（不需要穷举，5-8种即可）
- 重点标注"角色标志性表情"（如：反派的冷笑、女主的咬唇）

## 场景提取规则

### 必须提取的字段

| 字段 | 说明 |
|------|------|
| 场景名称 | 简短标识（如："女主的公寓客厅"） |
| 空间类型 | 室内/室外/半室外 |
| 具体设定 | 什么地方（办公室/街道/酒吧/古代庭院...） |
| 时间 | 白天/黄昏/夜晚/不固定 |
| 光照 | 自然光/灯光/混合/昏暗 |
| 氛围 | 温馨/压抑/奢华/简陋/... |

### 场景道具与背景

- **关键道具**：影响剧情的物品（如：放在桌上的合同、掉落的手机）
- **背景元素**：不影响剧情但构建氛围的元素（如：墙上的全家福、窗外的城市天际线）
- 区分关键道具和背景元素，前者需要单独建档，后者在场景描述中附带即可

---

## 道具提取规则

只为以下类型的道具单独建档：

1. **剧情道具** — 推动剧情的物品（信件、钥匙、武器、合同）
2. **角色标志物** — 标识角色身份的物品（项链、戒指、特殊武器）
3. **有状态变化的道具** — 在剧中发生变化的物品（完好→损坏、隐藏→暴露）

普通背景道具（杯子、书架、路灯）不需要单独建档。

---

## 状态变化跟踪

状态变化是一致性的重要维度。需要跟踪的变化类型：

| 变化类型 | 示例 |
|----------|------|
| 角色换装 | 日常装 → 晚礼服（第3集第2场） |
| 角色受伤 | 额头出血（第5集第4场起） |
| 角色变化 | 剪短头发（第8集起永久变化） |
| 场景时间 | 办公室白天 → 办公室夜晚 |
| 场景变化 | 公寓整洁 → 公寓被翻乱（第6集） |
| 道具状态 | 手镯完好 → 手镯断裂（第4集） |

每个变化必须标注：发生在哪一集哪个场景、是临时变化还是永久变化。

---

## 补全策略

剧本未明确描述的视觉细节，按以下优先级补全：

1. **从上下文推断** — 角色身份、职业、时代背景暗示的外观
2. **从题材惯例推断** — 同类型作品的常见设定（都市CEO通常穿西装）
3. **从 project-spec 风格推断** — 2D动漫风格下角色发色可以大胆，真人仿真则应写实
4. **标注为待确认** — 无法合理推断的，标注 `[待用户确认]`
