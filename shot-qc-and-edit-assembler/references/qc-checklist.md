# 质检维度与评分标准

> `shot-qc-and-edit-assembler` Step 1 读取本文件，确认质检的评分维度和标准。

---

## 五大质检维度

### 1. 角色一致性 (Character Consistency)

检查生成图像中的角色是否与 asset-bible 中的描述一致。

| 分数 | 标准 |
|------|------|
| 5 | 角色外貌完全匹配：发型、发色、面部特征、体型、服装全部正确 |
| 4 | 主要特征匹配，有 1 处细微偏差（如配饰颜色略偏） |
| 3 | 整体可辨认为同一角色，但有 2-3 处偏差（如发型略变） |
| 2 | 仅部分特征匹配，角色辨认度低 |
| 1 | 几乎无法辨认为目标角色，严重偏离描述 |

**关键检查项：**
- [ ] 发型和发色是否匹配
- [ ] 面部特征（眼型、肤色）是否匹配
- [ ] 服装是否是当前场景应穿的那一套（对照状态变化记录）
- [ ] 标志性特征（疤痕、配饰等）是否出现
- [ ] 体型比例是否一致

### 2. 构图匹配度 (Composition Match)

检查画面的景别、角色位置、视角是否符合 shot-plan 的要求。

| 分数 | 标准 |
|------|------|
| 5 | 景别、构图、角色位置完全符合 shot-plan 描述 |
| 4 | 景别正确，构图有轻微偏差（如角色偏左但 plan 要求居中） |
| 3 | 景别基本正确（如要求中景，实际偏近景），构图可接受 |
| 2 | 景别偏差较大（如要求特写，实际是全景） |
| 1 | 完全不符合 shot-plan 的构图要求 |

**关键检查项：**
- [ ] 景别是否与 shot-plan 指定的一致
- [ ] 角色站位是否符合描述
- [ ] 视角（平视/仰拍/俯拍）是否正确
- [ ] 重要元素是否都在画面安全区内

### 3. 情绪匹配度 (Emotion Match)

检查画面传达的情绪/氛围是否符合 shot-plan 的情绪标注。

| 分数 | 标准 |
|------|------|
| 5 | 色调、光影、表情完美传达目标情绪 |
| 4 | 整体情绪正确，细节上有优化空间 |
| 3 | 情绪方向正确但强度不足或稍有偏差 |
| 2 | 情绪模糊，无法明确感受到目标情绪 |
| 1 | 情绪完全错误（如要求悲伤但画面欢乐） |

**关键检查项：**
- [ ] 角色表情是否传达正确情绪
- [ ] 色调是否匹配情绪（暖色/冷色/灰调）
- [ ] 光影是否支持氛围（明亮/阴暗/柔和）
- [ ] 整体画面感觉是否与剧情此刻的情绪一致

### 4. 技术质量 (Technical Quality)

检查图像/视频是否存在 AI 生成的常见技术缺陷。

| 分数 | 标准 |
|------|------|
| 5 | 无可见技术缺陷 |
| 4 | 有 1 处轻微缺陷但不影响观看 |
| 3 | 有 2-3 处缺陷，可通过后期修复 |
| 2 | 有明显缺陷，部分影响观看体验 |
| 1 | 有严重缺陷（畸变肢体、面部崩坏），必须重生成 |

**常见技术缺陷清单：**
- [ ] 手指数量异常（多余手指/缺少手指）
- [ ] 肢体畸变或不自然弯折
- [ ] 面部崩坏（五官错位、不对称）
- [ ] 多余的肢体或身体部位
- [ ] 图像模糊或噪点过多
- [ ] 不自然的伪影或色块
- [ ] 出现不应有的文字或水印
- [ ] 背景穿帮或透视错误
- [ ] 物体边缘锯齿或融合不自然

### 5. 叙事连贯性 (Narrative Continuity)

检查当前镜头与前后镜头的视觉连贯性。

| 分数 | 标准 |
|------|------|
| 5 | 与前后镜头完美衔接，无视觉跳跃 |
| 4 | 基本连贯，有细微不一致但不出戏 |
| 3 | 存在可注意到的不一致，但剧情上可接受 |
| 2 | 明显的视觉断裂，影响叙事流畅 |
| 1 | 与前后镜头完全不连贯 |

**关键检查项：**
- [ ] 角色服装和状态是否与上一个镜头一致
- [ ] 场景环境是否保持一致（如光照方向、道具位置）
- [ ] 角色的空间位置是否与上一镜头的移动方向一致
- [ ] 情绪过渡是否自然

---

## 判定规则

| 判定 | 条件 |
|------|------|
| **通过 (Pass)** | 所有维度 ≥ 3 且无 critical 级缺陷 |
| **需微调 (Adjust)** | 有 1-2 项得 2 分，但可通过后期手段修复（裁剪、调色、局部修图） |
| **需重生成 (Redo)** | 任一维度得 1 分，或技术质量有 critical 缺陷，或 3 项以上得 2 分 |

## 严重程度定义

| 级别 | 定义 | 处理 |
|------|------|------|
| minor | 不影响观看，仅审美层面 | 建议优化但可不处理 |
| major | 会被多数观众注意到 | 必须处理（后期修复或重生成） |
| critical | 破坏画面质量或叙事逻辑 | 必须重生成 |
