# 剪辑合成建议规则

> `shot-qc-and-edit-assembler` Step 4 读取本文件，给出剪辑合成建议。

---

## 剪辑原则

### 1. 叙事优先

剪辑为叙事服务，不要为了炫技而加不必要的效果。每个转场、每个剪辑点都应该有叙事理由。

### 2. 呼吸感

给观众留呼吸空间。高潮之后必须有释放镜头。连续快节奏之后要有停顿。

### 3. 一致性

同一集内的剪辑风格保持统一。不要前半段用硬切，后半段突然用大量花式转场。

---

## 转场选择指南

| 场景变化类型 | 推荐转场 | 说明 |
|-------------|----------|------|
| 同场景内连续镜头 | 硬切 (Cut) | 最自然，不打断节奏 |
| 场景切换（同一时间线） | 硬切 或 短溶解 | 溶解时长 0.3-0.5s |
| 时间跳跃（小跳跃，几小时） | 溶解 (Dissolve) | 0.5-1s |
| 时间跳跃（大跳跃，数日/数年） | 淡出+淡入 (Fade) | 各 0.5-1s |
| 闪回/回忆 | 闪白 → 画面 → 闪白 | 闪白 0.1-0.3s |
| 梦境/幻想 | 柔焦溶解 | 加模糊过渡 |
| 高潮冲击点 | 闪白 或 闪黑 | 0.1-0.2s，强调冲击 |
| 悬念暂停 | 闪黑 | 留黑 0.5-1s |
| 情绪高点 → 立即反转 | 硬切 | 反差效果最强 |

## 配乐/音效节点规则

### BGM 切入/切出

| 时机 | 处理方式 |
|------|----------|
| 集开始 | 在第一个画面出现时同步切入，或延后 1-2s 先吸引视觉注意 |
| 情绪转折点 | BGM 转调或切换曲目，与画面转场同步 |
| 高潮前 | BGM 渐强，或突然静音（无声≈最大的声音） |
| 高潮点 | 鼓点/重音与画面冲击同步 |
| 结尾 | 淡出或在最后一帧硬切（悬念结尾用硬切更有效） |

### 音效建议

| 场景类型 | 建议音效 |
|----------|----------|
| 打巴掌/冲突 | 拍击音 + 短暂静音 |
| 震惊反应 | 低频嗡鸣 或 心跳声 |
| 回忆进入 | 回声/混响增加 |
| 悬念揭示 | 紧张弦乐刺音 (stinger) |
| 甜蜜时刻 | 轻铃音/钢琴单音 |
| 转场到下一场景 | 环境音渐变（室内→室外：加鸟鸣/车声） |

## 字幕/对白条规则

### 位置

| 画幅 | 字幕位置 | 避让 |
|------|----------|------|
| 竖屏 9:16 | 底部居中，距底边 350-400px | 避开平台互动按钮区 |
| 横屏 16:9 | 底部居中，距底边 80-120px | 避开弹幕区（B站上方 1/3） |

### 样式

- **字体**：无衬线中文字体（思源黑体/阿里巴巴普惠体）
- **大小**：竖屏 36-42px，横屏 28-34px
- **颜色**：白色字体 + 黑色描边(2px) 或 半透明黑底条
- **角色区分**：不同角色可用不同颜色名称标签（可选）

### 时机

- 字幕出现时机：比声音提前 0.1-0.2s（预读效应）
- 字幕消失时机：比声音结束延后 0.3-0.5s
- 单条字幕时长：最短 1s，最长 5s
- 超过 15 个字应分行或分条

## 片头/片尾处理

### 片头建议

| 方式 | 适用场景 |
|------|----------|
| 冷开场（无片头） | 竖屏短剧/漫剧的标准做法，直接进剧情 |
| 3 秒标题卡 | 横屏中长漫剧，标题+集数快速呈现 |
| 前情提要 | 连续剧情的中后期集数，2-3 个关键画面快速闪回 |

### 片尾建议

| 方式 | 适用场景 |
|------|----------|
| 悬念硬切 | 高潮未完直接黑屏，最强吊胃口 |
| 下集预告 | 3-5 秒快速剪辑下集关键画面 |
| 黑屏+文字 | "待续"/"To be continued" |
| 角色定格 | 最后一帧角色表情定住，配文字 |
